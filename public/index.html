Welcome to the COMP API

Version: 1.0.0

This API is developed and maintained by the COMP Junior Company at PUC Minas. You can find the project repository on GitHub.

Some Public Endpoints

  • GET All Members: Retrieve a list of all members. /api/members
  • POST Login: Authenticate using your credentials to access protected resources. /api/login
  • GET All Technologies: Retrieve a list of all available technologies. /api/technologies

To explore more features and endpoints, read the documentation and authenticate via the login endpoint.
